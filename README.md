# Juego de Ping Pong Clásico Desarrollado en Java

Un juego clásico de **Ping Pong (Pong)** desarrollado en **Java** utilizando las librerías **Swing (JFrame y JPanel)** para la interfaz gráfica.

Este proyecto implementa la lógica básica de un videojuego 2D, incluyendo movimiento de la pelota, colisiones, control mediante teclado, marcador y un bucle de juego (Game Loop).

---

##  Características del Juego 

-  Modo para dos jugadores.
-  Movimiento continuo de la pelota.
-  Colisiones con las paletas.
-  Rebotes contra los bordes superior e inferior.
-  Sistema de puntuación.
-  Controles mediante teclado.
-  Interfaz gráfica desarrollada con Java Swing.
-  Game Loop ejecutado mediante un hilo (`Thread`).

## Controles del Juego

### Jugador 1 (Izquierda)

| Tecla | Acción |
|-------|--------|
| W | Mover hacia arriba |
| S | Mover hacia abajo |

### Jugador 2 (Derecha)

| Tecla | Acción |
|-------|--------|
| ↑ | Mover hacia arriba |
| ↓ | Mover hacia abajo |

##  Cómo ejecutar el proyecto

1. Clona el repositorio.

```bash
git clone https://github.com/julioromero015/Juego_Pong.git
```

2. Entra al proyecto.

```bash
cd Juego_Pong
```

3. Compila el programa.

```bash
javac JuegoPong.java
```

4. Ejecuta el juego.

```bash
java JuegoPong
```
